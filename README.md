SkyTab Venue - Feature Highlights

Shift4's SkyTab Venue is a high-volume point-of-sale platform for stadiums and arenas.

Core Platform
Performance: Optimized for burst traffic and high transaction volume.
Architecture: Hybrid cloud model with offline-resilient APKs.
Payments: Native EMV, NFC, Tokenization, and Digital Wallets.
UX: Simplified interface for rapid staff training.

Hardware Support
Fixed POS: PAX L1400 & A3700 (Concessions/Bars).
Mobile: PAX A800 (In-Seat/Hawking).
Kiosk: Elo Tablet + Olea Stand (Self-Ordering).

Web Applications Suite
Menu Manager: Revenue centers, items, modifiers, taxes.
Orders: Search, review, and refund transactions.
Reports: Financial analytics, exports, and scheduling.
Loyalty: Rewards program management and user lookup.
Devices: Printer routing and terminal configuration.
Access: User roles and permissions management.

Mobile Wallet
Identity: SSO with Ticketmaster, SeatGeek, AXS.
Passes: Apple & Google Wallet support (QR/NFC).
Payments: Card-on-file and stored value.
Engagement: Push notifications and geofenced lock screen access.

Ecosystem Integrations
Ticketing: Real-time validation.
Inventory: Two-way sync with YellowDog.
Menu Boards: Dynamic digital pricing.
Autonomous Retail: Mashgin and Zippin support.
